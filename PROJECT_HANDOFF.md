# PathPilot - Project Handoff

## Project Overview

PathPilot is an AI-powered learning platform designed to take students from beginner to job-ready through structured roadmaps, AI tutoring, quizzes, progress tracking, and gamification.

## Tech Stack

* Frontend: React + TypeScript + Tailwind + Vite
* Backend: Node.js + Express
* Database: MongoDB
* AI: Gemini API
* Payments: Razorpay
* Auth: JWT + Google OAuth
